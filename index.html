Rock Paper Scissors Robotic Hand

An intelligent Arduino-based robotic system that plays Rock Paper Scissors using programmable servo motors, mechanical engineering principles, and embedded systems.

Project Description

This project is a robotic hand powered by Arduino that can automatically play Rock, Paper, Scissors. It uses servo motors to simulate finger movements and mechanical joints to replicate real hand gestures.

It demonstrates robotics, automation, embedded systems programming, and mechanical design in a practical educational model suitable for STEM learning.

Many students learn robotics only theoretically. This project creates a real interactive robotic prototype that shows how electronics and mechanics work together.

Functional Requirements

The robotic hand closes all fingers to simulate the Rock gesture.

The robotic hand opens all fingers fully to simulate Paper.

The robotic hand extends two fingers while closing the others to simulate Scissors.

The built-in buzzer produces sound effects during gesture activation.
